Shinra Omega Radio 8.0.0 — Serenity Rebuild

- Evolution, rituals, scanner locks and rare events use gentle world blooms rather than sharp synthetic hits.
- Slow 8–13 second layer fades, more silence, softer presence and restrained high-frequency energy.
- New comfort master with 48 kHz context, 3.15 kHz presence contour, soft air shelf, 15.8 kHz safety filter and slow musical dynamics.
- SERENITY is the default listening profile. DEEP FOCUS, DEEP REST and IMMERSIVE remain available.
